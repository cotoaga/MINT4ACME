We believe the approach we have chosen to have several immediate benefits in terms of:

  • Applications functionality, since the Internet is the largest market for software developers, creating an environment where many of the complex object manipulation routines we will need to fully support the multimedia aspects of MINT can be bought as OTC (over The Counter) software.
  • User interface standardization that has been created to support much of the world's population, and has been demonstrated to be acceptable to and accepted by even poorly competent computer users.
  • Business functionality, since the use of the Internet paradigm will enable dealers and customers to be largely (and ultimately totally) independent from the physical location of the user, of the application and of the data.
  • Maintenance costs, since the time and effort required to introduce a new model would be minimal provided that existing HTML structures and objects were used, so that only multimedia object contents and text would need to be changed if necessary.
