Is my browser capable of viewing all of MINT?

This depends on some points. The Browser itself, activated features, add-ons and the installed Plug-ins. The following secions will give you all answers. If everything is fine you should see only green check marks and no red cross.

Browser

MINT for ACME can be used only with the latest versions of the Netscape and Microsoft browsers. This means you need at least the Netscape Communicator V4.01 or the Microsoft Internet Explorer V4.0 Preview2 in order to view all of MINT for ACME. You can click on the icons in the table to get the browser you prefer.

For the Netscape Communicator these features can be enabled from the preferences dialog. You can reach this dialog by choosing the menupoint edit/preferences... where you must select the advanced settings.

If you are using Microsoft Internet Explorer you must go to the options dialog. You can reach this dialog by choosing the menupoint view/options.... In the security section of the advanced tab page you can switch Cookies on and off. In the security tab page you can activate Java by choosing custom security and enabling Java in the corresponding section.

At the end of all these features on last general advice! MINT for ACME will terrible stress your browser environment. So please be kind and give it some time to execute all the tasks.

Tune up network performance

Due to the great number of big multimedia objects you will need at least an ISDN based access to the internet. However, MINT for ACME is Java BOOSTER optimized and there is a simple way to boost up access speed - you can even work offline!



By using Innovative Software's Java BOOSTER (it's free!) you can permanently cache all MINT for ACME objects and reduce network traffic to a minimum. All you have to do is to follow the install instructions, to browse on time through the entire document and finaly to mark MINT for ACME as an non updateable favorite.

How should I use this hypermedia document?

Click on the MINT for ACME logo to start from the beginning. Use the forward and back buttons on the top and bottom of each page to browse through the document.


MINT for ACME offers special guided tours through its features. Choose the corresponding tour from the drop down box. Click the leftmost button to start from the beginning and then flip through with the forward and back buttons. This control can be undocked if you wish to customize your display area.


Use the tree control on the left hand side to explore the struture of the document. You can expand and contract sections by clicking the green buttons. By clicking the red buttons you can jump directly to the identified sub-sections.
