Digital Archaeology Project

This restored version was created as a digital archaeology project in 2025, with the goal of preserving this pioneering web application from 1997 while making it accessible to modern browsers.

Restoration Team

  • Project Lead: Digital Archaeology Initiative
  • Technical Restoration: Web Preservation Group
  • Historical Research: Computing History Archive

Special thanks to the original creators for developing this innovative application that was years ahead of its time.

Digital Archaeology Note: This is a modern restoration of a 1997 web application. The original used Flash animations, auto-playing WAV files, frames, and obsolete web technologies. These have been replaced with HTML5 equivalents while preserving the original look and navigation structure. Background images are now more visible, and redundant copyright footers have been removed.
