4.1. Technical Environment

The MINT framework relies on a sophisticated technical environment to deliver its business functionality. This section describes the technologies employed in the framework and their integration into a cohesive solution.

The technical environment of the MINT framework is designed to accommodate the diverse requirements of automotive sales, from rich multimedia presentation to offline operation to integration with existing dealer and manufacturer systems.

Technical Stack Overview

Client Technologies

  • Web Browsers: Netscape Navigator 4.0 or Internet Explorer 4.0
  • Client-Side Scripting: JavaScript 1.1
  • Multimedia: Macromedia Flash, QuickTime, VRML
  • Data Synchronization: Proprietary replication mechanism

Server Technologies

  • Web Server: Microsoft Internet Information Server, Netscape Enterprise Server
  • Application Server: MINT Application Framework
  • Database: Oracle, Microsoft SQL Server
  • Integration: MINT Integration Services

Network Requirements

  • Online Operation: TCP/IP connectivity to MINT servers
  • Offline Operation: Local data replicas with synchronization capability
  • Dealer Integration: Connectivity to dealer management systems
  • Manufacturer Integration: Secure connections to ACME production systems
