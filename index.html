About This Project

MINT (Multimedia Information, Negotiation and Transaction) for ACME was a revolutionary web-based automotive sales application developed in 1997. It featured:

  • A sophisticated four-layer architecture
  • Rich multimedia experiences
  • Offline capabilities (years before Progressive Web Apps)
  • Frameset-based navigation
  • Integration with legacy automotive systems

This digital archaeology project preserves the original application while making it accessible to modern viewers. It showcases web development practices from 1997 alongside modern equivalents.
