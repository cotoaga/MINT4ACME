About the Project

MINT for ACME was a revolutionary web-based automotive sales application developed in 1997 by CSC PLOENZKE AG and INC. InterNet Consult GmbH. It featured:

  • A four-layer architecture (Browser, Replication Server, Application Server, Database)
  • Offline capabilities years before Progressive Web Apps
  • Rich multimedia content through various plugins and technologies
  • Integration with legacy automotive systems
  • An iterative development approach resembling Agile methods before the Agile Manifesto

This revival project aims to preserve this piece of web history while making it accessible to modern users through current web technologies.

The "Then vs. Now" Approach

As we modernize each section, we'll provide side-by-side comparisons showing how the original implementation compares to modern web development practices. This will showcase both how far web technology has come and how visionary the original project was for its time.
